## docker-compose php:5.6.40 + mysql:5.6 + nginx:latest environment

## 一、Common commands

1、boot image：
```
docker-compose up -d
```
2、down image：
```
docker-compose down
```
3、restart image：
```
dokcer-compose restart
```
4、build：
```
docker-compose build
```
4、update Dockerfile after build：
```
docker-compose up -d --build
```
